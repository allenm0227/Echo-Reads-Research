# Data Science Research Group App

This repository hosts the design and documentation for a mobile/web app created for a Data Science Research Group. The app is designed to help students and researchers track experiments, organize datasets, and visualize results in an accessible way.

## Overview

- **Role:** App Designer & Developer  
- **Tech/Tools:** Figma/Canva for UI design (prototype), HTML/CSS/JS (front-end), GitHub for version control  
- **Goals:**  
  - Centralize research notes and experimental logs  
  - Make data summaries easy to view and share  
  - Provide a clean, student-friendly interface

## Features (planned / prototyped)

- Dashboard for ongoing research projects  
- Experiment detail pages with notes, attachments, and tags  
- Simple data visual cards for quick insights  
- User-friendly interface based on the Canva prototype
